HTML

  • The head element contains information about the webpage
  • The meta element contains information about the webpage
  • The body element represents the visible content shown to the user
  • h1 - h6 are identifiers that allow the developer to create a heirarchy of headers

CSS

  • A margin inducates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Designating what property you want by the keyword will act as a global style.
  • if using a css external doc need to make sure to link in the head section after meta properties.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: add all new files
  • git commit -M: leave a message with the added files

JavaScript

  • A variable is aname container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • An array is a variable that is able to store mulitple variables
  • The js doc will be linked using the "script" keyword after the closing footer and before the closing body
